Administrator’s Guide

Administrator’s Guide

This guide is intended for any user who assumes role of a developer/administrator of a Teiid Spring Boot instance.

Before one can delve into Teiid Spring Boot it is very important to learn few basic constructs of Teiid Spring Boot, like what is VDB? what is Model? etc. For that please read the short introduction.
